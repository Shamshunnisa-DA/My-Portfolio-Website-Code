OTT Platforms Data Analysis

Shamshunnisa M Power BI Project

Project Overview

Objective

To analyze all aspects of different OTT platforms and their growth patterns, content distribution, and viewer preferences.

Dataset

Collected data for 3 major platforms (Netflix, Amazon Prime, Disney+ Hotstar) from Kaggle & GitHub in CSV format with 7-8 key metrics per dataset.

Tools Used

Microsoft Power BI for data transformation, visualization, and dashboard creation.

Methodology

1

Data Import

Downloaded datasets from Kaggle & GitHub references and imported CSV files into Power BI for analysis.

2

Data Transformation

  • Cleaned all null values and handled missing data
  • Added calculated columns for enhanced analysis
  • Removed duplicate entries
  • Standardized formats across datasets
3

Visualization

  • Pie charts for content distribution
  • Stacked bar charts for comparative analysis
  • Donut charts for rating breakdowns
  • Flash cards for key metrics
  • Slicers for interactive filtering
4

Dashboard Design

  • Organized visualizations in logical sequence
  • Added explanatory cards for key numbers
  • Implemented cross-filtering functionality
  • Ensured mobile-responsive layout

Key Insights

Netflix

  • 15K content pieces dominated by movies (12K), with series gaining awards momentum post-2016
  • Movies have won more awards overall despite series growth
  • Viewer preference leans toward longer movies
  • "R" ratings dominate at 45%, reflecting mature audience focus

Amazon Prime

  • Hosts 9,684 shows with movies (7,814) as majority content
  • U.S. dominates content distribution at 93.06%
  • Viewer ratings lean toward 13+ (29.59%)
  • Indian content remains small at just 2.61% of total

Disney+ Hotstar

  • 1,450 shows with movies (1,052) as primary content
  • Solid average IMDb score of 6.8 across content
  • Focus on diverse content genres and family-friendly programming
  • TV-14 and TV-PG ratings dominate the platform
  • Notable yearly content additions from 2019 to 2021
