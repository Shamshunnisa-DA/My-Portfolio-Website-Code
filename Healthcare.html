Health Care Data Analysis

Shamshunnisa M Power BI Project

Project Overview

Objective

To analyze inpatient and outpatient waiting lists over 3 years (2018-2022) to identify trends and bottlenecks in healthcare service delivery.

Dataset

Patient data from 2018–2022 in CSV format containing:

  • Inpatient data: 9 key columns
  • Outpatient data: 10 key columns

Tools Used

Microsoft Power BI for end-to-end data processing, visualization, and interactive reporting.

Methodology

1. Data Import

Downloaded datasets from YouTube reference sources and imported CSV files into Power BI Desktop for analysis.

2. Data Transformation

  • Cleaned null/missing values across both datasets
  • Added calculated columns for enhanced analysis
  • Standardized date formats across years
  • Created relationships between inpatient/outpatient tables

3. Visualization

Pie Chart: Outpatient vs Inpatient
Stacked Bar: Specialty Wait Times
Line Graph: 3-Year Trends

4. Dashboard Design

  • Organized visualizations in logical clinical workflow sequence
  • Added KPI cards for critical wait time metrics
  • Implemented cross-filtering by specialty, age group, and year
  • Mobile-responsive layout for clinical staff access

Key Insights

Waitlist Volume Trends

The total wait list increased by 10.8% year-over-year, with outpatient cases dominating (74.36% of total cases).

Specialty-Specific Findings

Dermatology and Accident & Emergency specialties show the longest wait times (avg. 6-8 months), while Pain Relief departments maintain the shortest (under 3 weeks).

Age Group Analysis

Patients aged 16–64 face the longest waiting periods (over 18 months), while those 65+ generally wait under 3 months due to priority pathways.

Outpatient Dominance

Outpatient numbers consistently exceed Day Cases and Inpatients, fluctuating between 502K–629K over the 3-year period (July 2018–Jan 2021).
